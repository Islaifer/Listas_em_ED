Este repositório tem o intuito de exemplificar as estruturas de dados de lista simplesmente
encadeada e duplamente encadeada na linguagem java.
O caso proposto foi um sistema que armazena o nome e a media de alunos, o intuito foi fazer
uma lista simplesmente encadeada e uma lista duplamente encadeada que armazene, delete, mostre a lista
e ordene ela por ordem de melhores médias.
